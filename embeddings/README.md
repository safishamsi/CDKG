# Embeddings Directory

This directory will be automatically populated when you run the pipeline.

## Contents (After Running Pipeline)

- `speaker_embeddings.npy` - Speaker embeddings
- `speaker_metadata.json` - Speaker metadata
- `talk_embeddings.npy` - Talk embeddings
- `talk_metadata.json` - Talk metadata
- `tag_embeddings.npy` - Tag embeddings
- `tag_metadata.json` - Tag metadata
- `event_embeddings.npy` - Event embeddings
- `event_metadata.json` - Event metadata
- `category_embeddings.npy` - Category embeddings
- `category_metadata.json` - Category metadata
- `all_embeddings.npy` - Combined embeddings
- `index_mapping.json` - Index to metadata mapping
- `faiss_index.bin` - FAISS vector index
- `faiss_index_info.json` - Index information

## Size

Approximately 1.4 MB total after generation.
