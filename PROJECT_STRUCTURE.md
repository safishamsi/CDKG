## Quick Start

1. **Read**: `START_HERE.md`
2. **Setup**: Follow `QUICKSTART.md`
3. **Configure**: Copy `.env.template` to `.env` and fill in
4. **Add Data**: Copy `cdl_db` to `data/cdl_db/`
5. **Run**: `python run_pipeline.py`
6. **Use**: Import `RAGSystem` and start querying!
